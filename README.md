# 🪴 POT2POT (팟투팟) - 반려식물 케어 & 아카이빙

**POT2POT(팟투팟)**은 AI 기술을 결합하여 반려식물의 상태를 직관적으로 기록하고 관리하는 **반려식물 케어 및 성장 아카이빙 모바일 웹 서비스**입니다. 
식물 사진 촬영 한 번으로 배경을 깔끔하게 제거(누끼)해 나만의 다이어리 스티커로 변환하고, Gemini AI 분석을 통해 품종 정보와 물주기 주기를 자동으로 추천받아 기록할 수 있습니다.

---

## 🚀 핵심 기능 (Key Features)

### 1. Smart BG Remove & Sticker Outline (배경 제거 & 스티커화)
* **다이나믹 코너 샘플링**: 이미지의 4개 모서리 색상을 동적으로 분석해 배경색을 탐색 및 지워 줍니다.
* **고립 노이즈 필터 (BFS Island Filter)**: 메인 식물 이외의 자잘한 상태 표시줄 텍스트나 고립된 먼지 픽셀을 탐색하여 깔끔히 한 덩어리로 잘라냅니다.
* **스티커 스타일 드로잉**: 도려낸 이미지 외곽을 따라 8px 두께의 흰색 테두리(`#ffffff`)와 부드러운 입체 드롭 섀도우를 가미하여 귀여운 다이어리 스티커 비주얼을 선사합니다.

### 2. Gemini AI Plant Analysis (Gemini 기반 식물 자동 분석)
* **무료 AI 스캐닝**: Google AI Studio의 무료 **Gemini 1.5 Flash API**를 클라이언트 사이드에서 활용합니다.
* **자동 프로필 생성**: 사진을 기반으로 식물의 **정확한 품종(Species)**을 감지하고, 어울리는 **귀여운 별명(Nickname)**을 지어 주며, **권장 물주기 주기(Water Interval)**를 추천하여 입력 폼을 자동으로 채워 줍니다.
* **개인정보 및 키 보안**: API Key는 브라우저 내부의 `localStorage`에만 저장되며 외부 서버로 전송되지 않습니다.

### 3. Care Timeline & Mini Calendar (케어 타임라인 & 캘린더)
* **성장 다이어리**: 입양일(🌱), 물주기(💧), 영양제(💊), 분갈이(🪴) 등 반려식물과 함께한 성장 스토리를 세로형 타임라인에 아름답게 아카이빙합니다.
* **미니 캘린더**: 캘린더 탭에서 날짜별 케어 내역을 직관적인 도트 아이콘 마커로 시각화하여 한달간의 정성을 한눈에 모아볼 수 있습니다.

### 4. Sleek Dark Theme (형광 라임 다크 모드)
* **명도 대비 최적화**: 깊은 차콜 블랙 배경(`#222222`)과 화제성 있는 형광 라임 네온(`#CDFF62`)을 조합하여 세련되고 모던한 정원을 묘사했습니다.
* **모바일 편의성**: 터치 오류 방지를 위해 주요 버튼 크기를 1.3배 이상 확대 배치하고, 미니멀한 텍스트 배치를 적용했습니다.

---

## 🛠 기술 스택 (Tech Stack)

* **프론트엔드**: HTML5, CSS3 (Custom Variables), Vanilla JavaScript (ES6+)
* **AI API**: Google Gemini 1.5 Flash API (Client-side JSON mode)
* **개발 서버**: Python 3 (Simple HTTP Server)

---

## 💻 실행 방법 (How to Run)

### 1. 로컬 개발 서버 구동
프로젝트 루트 폴더에서 아래 명령어를 실행하여 웹 서버를 켭니다.
```bash
python3 -m http.server 8000
```

### 2. 브라우저 접속
* **PC 브라우저**: [http://localhost:8000](http://localhost:8000)
* **모바일 브라우저**: PC와 모바일 기기가 같은 와이파이(로컬 네트워크)에 연결되어 있을 때 PC의 로컬 IP(예: `http://192.168.117.22:8000`)로 접속하여 미리 볼 수 있습니다.

### 3. Gemini API 키 설정
서비스 내 **설정(⚙️)** 탭으로 이동하여 구글 AI Studio에서 무료로 발급받은 API 키를 입력하면 AI 식물 분석 기능을 즉시 테스트하실 수 있습니다.
